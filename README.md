## Coding style

```sh
swift-format format -i -r ./
swift-format lint -i -r ./
```

If you want to do formatting in XCode, install XCFormat extension for XCode.

## Tracking

Sentry:

```sh
brew install getsentry/tools/sentry-wizard && sentry-wizard -i ios
```
